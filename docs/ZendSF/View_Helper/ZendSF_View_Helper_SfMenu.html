ZendSF_View_Helper_SfMenu

Description
Description | Vars (details) | Methods (details)

This class can add a 'span' tag to the 'a' tag when specified in the config file using a 'span' element and setting it to 1. If no 'span' element is set or it is set to 0 then the 'span' element is not generated unless a href is not specified.

example:

  1.  <?xml version="1.0" encoding="UTF-8"?>
  2.  <nav>
  3.   <topMenu>
  4.    <home>
  5.     <label>Home</label>
  6.     <span>1</span>
  7.     <module>core</module>
  8.     <controller>index</controller>
  9.     <action>index</action>
  10.    </home>
  11.   </topMenu>
  12.  </nav>
