		_checkDate (line 216)
	
 
	
	

Checks the date format for the locale.


If locale is set to en_GB then entering a en_US format will throw an error.  eg. 
  locale = en_GB 
  date = '2011/01/01' throws error 
  date = '01/01/2011' returns true
